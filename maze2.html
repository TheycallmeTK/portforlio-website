Features

Using Unity and C# scripting, I created a grid of walls that, on start of the game, run an algorithm and do on of three things:

-disappear

-disappear after the given time

-move in one of two directions after the given time

The walls are set on a timer, and upon reaching that interval they run the algorithm again and either move or disappear, making the maze random on every start and always changing.
