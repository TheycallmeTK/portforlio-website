Gravity Snake

Using Visual Studio C++, Box 2D physics, and SFML graphics, this is a 2D game where you control a circle while it's falling with the W, A, S and D keys in order to hit the target at the bottom. You can choose the amount of targets to set up.

Score calculated by taking input, less input yields a bigger score.
