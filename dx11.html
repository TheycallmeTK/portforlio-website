About

A simple infinite runner game using third party textures and models implemented into a custom-made graphics engine using DX11 and C++ programming.

Use the A and D keys to switch lanes and avoid oncoming obstacles as you gain points. The player speed increases as you travel further, increasing the difficulty.

The high score for each session is saved and displayed at the top right of the screen.

This engine was a class project created over the course of a semester. This engine utilizes DirectX 11 Libraries to implement features including physically based rendering, lighting/shadows, and normal mapping.
